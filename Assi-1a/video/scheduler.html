Job Scheduler using Priority Queue (Min Heap)

Problem Statement

A software company is developing a new job scheduler that efficiently manages different processes. The goal is to minimize average waiting time by giving priority to shortest time tasks first using a Min Heap (Priority Queue).

Concept of Job Scheduling

Job Scheduling is a technique used by operating systems to decide the order in which processes are executed by the CPU. Each process has a certain execution time and the scheduler decides which process should run first.

To minimize the average waiting time, the Shortest Job First (SJF) strategy is used. In this assignment, SJF is implemented using a Min Heap Priority Queue, where the job with the smallest execution time gets the highest priority.

Why Priority Queue?

A Priority Queue stores elements in such a way that the element with the highest priority is always removed first.

  • Min Heap → Shortest execution time first
  • Max Heap → Longest execution time first
  • Efficient insertion and deletion: O(log n)
  • Efficient retrieval of highest priority job: O(1)

Job Structure

Represents a process with an ID and execution time. 


class Job {
    int id;
    int time;

    Job(int id, int time){
        this.id = id;
        this.time = time;
    }
}

Min Heap (Priority Queue)

Stores jobs in ascending order of execution time. The job with smallest time is executed first.

Smaller execution time → higher priority


PriorityQueue minHeap = new PriorityQueue<>(
    (a, b) -> a.time - b.time
);

Insert Job

Inserts a new job into the scheduler. 


void addJob(int id, int time){
    minHeap.add(new Job(id, time));
}

Execute Jobs

Removes jobs one by one from the priority queue and calculates the average waiting time. 


void executeJobs(){
    int wait = 0, totalWait = 0, n = minHeap.size();

    while(!minHeap.isEmpty()){
        Job j = minHeap.poll();
        System.out.println("Executing Job " + j.id + " Time: " + j.time);
        totalWait += wait;
        wait += j.time;
    }

    System.out.println("Average Waiting Time: " + (float)totalWait / n);
}

Complete Program (With Comments)

Provides a menu driven interface to insert and execute jobs. 


import java.util.*;

// Job class represents each process
class Job {
    int id, time;

    // Constructor
    Job(int id, int time){
        this.id = id;
        this.time = time;
    }
}

public class JobScheduler {

    // Min Heap Priority Queue (Shortest Job First)
    static PriorityQueue minHeap = new PriorityQueue<>(
        (a, b) -> a.time - b.time
    );

    // Add job to scheduler
    static void addJob(int id, int time){
        minHeap.add(new Job(id, time));
    }

    // Execute jobs and calculate average waiting time
    static void executeJobs(){
        int wait = 0;        // Waiting time for current job
        int total = 0;       // Total waiting time
        int n = minHeap.size();

        while(!minHeap.isEmpty()){
            Job j = minHeap.poll();           // Get highest priority job
            System.out.println("Executing Job " + j.id + " Time: " + j.time);

            total += wait;                    // Add waiting time
            wait += j.time;                  // Increase waiting time
        }

        System.out.println("Average Waiting Time: " + (float) total / n);
    }

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int ch, id, t;

        do{
            System.out.println("\n1.Add Job  2.Execute Jobs  0.Exit");
            ch = sc.nextInt();

            if(ch == 1){
                System.out.print("Job ID: ");
                id = sc.nextInt();
                System.out.print("Execution Time: ");
                t = sc.nextInt();
                addJob(id, t);
            }
            else if(ch == 2){
                executeJobs();
            }

        }while(ch != 0);
    }
}
