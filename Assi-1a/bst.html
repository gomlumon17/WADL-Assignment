Introduction

A Binary Search Tree (BST) is a hierarchical data structure in which each node contains a key value and two child pointers, namely the left child and the right child. The BST follows an important property:

For every node in the tree, all values in the left subtree are smaller than the node’s value, and all values in the right subtree are greater than the node’s value.

Due to this property, Binary Search Trees allow efficient operations such as insertion, deletion, and searching. BSTs are widely used in applications that require ordered data storage and fast retrieval.

Explanation of Binary Search Tree Program

This section explains the complete flow of the Binary Search Tree (BST) program implemented in Java. The explanation is provided step-by-step, starting from data definition to performing various BST operations.

Step 1: Defining the Node Structure

Each node of the Binary Search Tree contains three parts: data, reference to the left child, and reference to the right child. 


class Node {
    int data;          // Value stored in the node
    Node left, right;  // References to left and right child nodes

    // Constructor to initialize a new node
    Node(int value) {
        data = value;
        left = right = null;
    }
}

The Node class represents a single element of the BST. Initially, both child references are set to null.

Step 2: Creating the Binary Search Tree Class

The BinarySearchTree class contains the root node and all the operations performed on the tree. 


class BinarySearchTree {
    Node root;

    // Constructor
    BinarySearchTree() {
        root = null;
    }
}

The root node acts as the starting point of the Binary Search Tree. Initially, the tree is empty.

Step 3: Inserting a Node (Handling Duplicate Values)

Insertion is performed recursively by comparing the value to be inserted with the current node. Duplicate values are not allowed in the BST. 


Node insert(Node root, int value) {

    // If tree is empty, create a new node
    if (root == null) {
        return new Node(value);
    }

    // If value is smaller, insert in left subtree
    if (value < root.data) {
        root.left = insert(root.left, value);
    }
    // If value is greater, insert in right subtree
    else if (value > root.data) {
        root.right = insert(root.right, value);
    }
    // Duplicate values are ignored
    else {
        System.out.println("Duplicate value not allowed");
    }

    return root;
}

If the value already exists, the insertion is skipped to avoid duplicates.

Step 4: Searching an Element

Searching is done by comparing the key with the current node’s data. 


boolean search(Node root, int key) {

    // Base case: tree is empty or key is found
    if (root == null) {
        return false;
    }

    if (root.data == key) {
        return true;
    }

    // Search in left or right subtree
    if (key < root.data) {
        return search(root.left, key);
    } else {
        return search(root.right, key);
    }
}

The function returns true if the element is found, otherwise false.

Step 5: Tree Traversals

Traversals are used to display the elements of the tree. Inorder traversal prints the BST elements in sorted order. 


void inorder(Node root) {
    if (root != null) {
        inorder(root.left);     // Visit left subtree
        System.out.print(root.data + " ");
        inorder(root.right);    // Visit right subtree
    }
}

Step 6: Finding the Depth (Height) of the Tree

Depth of a tree is the maximum number of nodes from the root to the leaf node. 


int depth(Node root) {
    if (root == null) {
        return 0;
    }

    int leftDepth = depth(root.left);
    int rightDepth = depth(root.right);

    return Math.max(leftDepth, rightDepth) + 1;
}

Step 7: Creating Mirror Image of the Tree

In mirror image, left and right child nodes are swapped recursively. 


void mirror(Node root) {
    if (root == null) {
        return;
    }

    Node temp = root.left;
    root.left = root.right;
    root.right = temp;

    mirror(root.left);
    mirror(root.right);
}

Step 8: Level Order Traversal

Level order traversal displays nodes level-by-level using a queue. 


void levelOrder(Node root) {
    if (root == null) return;

    Queue q = new LinkedList<>();
    q.add(root);

    while (!q.isEmpty()) {
        Node current = q.poll();
        System.out.print(current.data + " ");

        if (current.left != null)
            q.add(current.left);
        if (current.right != null)
            q.add(current.right);
    }
}

This completes the explanation of the Binary Search Tree implementation and its operations in Java.

Complete Binary Search Tree Program (Java)

The following is a complete, menu-driven Java program that implements a Binary Search Tree and performs operations such as insertion, deletion, searching, traversal, depth calculation, and mirror image creation.

Complete Java Code


import java.util.*;

class Node {
    int data;
    Node left, right;

    Node(int value) {
        data = value;
        left = right = null;
    }
}

class BinarySearchTree {
    Node root;

    BinarySearchTree() {
        root = null;
    }

    // Insert node (duplicates not allowed)
    Node insert(Node root, int value) {
        if (root == null)
            return new Node(value);

        if (value < root.data)
            root.left = insert(root.left, value);
        else if (value > root.data)
            root.right = insert(root.right, value);
        else
            System.out.println("Duplicate value not allowed");

        return root;
    }

    // Search operation
    boolean search(Node root, int key) {
        if (root == null)
            return false;

        if (root.data == key)
            return true;

        if (key < root.data)
            return search(root.left, key);
        else
            return search(root.right, key);
    }

    // Inorder traversal
    void inorder(Node root) {
        if (root != null) {
            inorder(root.left);
            System.out.print(root.data + " ");
            inorder(root.right);
        }
    }

    // Find minimum value node
    Node minValueNode(Node root) {
        Node current = root;
        while (current.left != null)
            current = current.left;
        return current;
    }

    // Delete operation
    Node delete(Node root, int key) {
        if (root == null)
            return root;

        if (key < root.data)
            root.left = delete(root.left, key);
        else if (key > root.data)
            root.right = delete(root.right, key);
        else {
            // Node with one or no child
            if (root.left == null)
                return root.right;
            else if (root.right == null)
                return root.left;

            // Node with two children
            Node temp = minValueNode(root.right);
            root.data = temp.data;
            root.right = delete(root.right, temp.data);
        }
        return root;
    }

    // Depth of tree
    int depth(Node root) {
        if (root == null)
            return 0;
        return Math.max(depth(root.left), depth(root.right)) + 1;
    }

    // Mirror image
    void mirror(Node root) {
        if (root == null)
            return;

        Node temp = root.left;
        root.left = root.right;
        root.right = temp;

        mirror(root.left);
        mirror(root.right);
    }

    // Level order traversal
    void levelOrder(Node root) {
        if (root == null)
            return;

        Queue q = new LinkedList<>();
        q.add(root);

        while (!q.isEmpty()) {
            Node current = q.poll();
            System.out.print(current.data + " ");

            if (current.left != null)
                q.add(current.left);
            if (current.right != null)
                q.add(current.right);
        }
    }

    public static void main(String[] args) {
        BinarySearchTree bst = new BinarySearchTree();
        Scanner sc = new Scanner(System.in);
        int choice, value;

        do {
            System.out.println("\n--- Binary Search Tree Menu ---");
            System.out.println("1. Insert");
            System.out.println("2. Delete");
            System.out.println("3. Search");
            System.out.println("4. Inorder Traversal");
            System.out.println("5. Depth of Tree");
            System.out.println("6. Mirror Image");
            System.out.println("7. Level Order Traversal");
            System.out.println("0. Exit");
            System.out.print("Enter choice: ");

            choice = sc.nextInt();

            switch (choice) {
                case 1:
                    System.out.print("Enter value to insert: ");
                    value = sc.nextInt();
                    bst.root = bst.insert(bst.root, value);
                    break;

                case 2:
                    System.out.print("Enter value to delete: ");
                    value = sc.nextInt();
                    bst.root = bst.delete(bst.root, value);
                    break;

                case 3:
                    System.out.print("Enter value to search: ");
                    value = sc.nextInt();
                    if (bst.search(bst.root, value))
                        System.out.println("Element found");
                    else
                        System.out.println("Element not found");
                    break;

                case 4:
                    System.out.print("Inorder Traversal: ");
                    bst.inorder(bst.root);
                    System.out.println();
                    break;

                case 5:
                    System.out.println("Depth of Tree: " + bst.depth(bst.root));
                    break;

                case 6:
                    bst.mirror(bst.root);
                    System.out.println("Mirror image created");
                    break;

                case 7:
                    System.out.print("Level Order Traversal: ");
                    bst.levelOrder(bst.root);
                    System.out.println();
                    break;
            }

        } while (choice != 0);

        sc.close();
    }
}
